# Campus Query Chatbot – IILM

A college-specific enquiry chatbot built as part of my '1st semester AI/ML academic project'.  
The goal was to provide quick answers to basic campus-related questions, as no such chatbot existed on the IILM website.

---

# 🔧 Tech Stack
- Python, Flask  
- HTML, CSS, JavaScript  
- Pandas  
- TF-IDF & Cosine Similarity  
- CSV-based data storage  

---

# 🧠 How It Works
- User queries are matched against a predefined FAQ dataset using TF-IDF and cosine similarity.
- If a suitable match is found, the corresponding answer is returned.
- If not, a fallback response is shown and the unanswered question is stored in a separate CSV file for future updates.

---

# ✨ Features
- College-specific enquiry chatbot for IILM students  
- NLP-based question matching  
- Fallback handling for unknown queries  
- Easy to extend by updating CSV files  

---

Academic prototype

# ⚠️ Scope
Currently limited to college-specific FAQs stored in CSV files.
